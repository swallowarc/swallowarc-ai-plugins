- 記事末尾に見出し「## 参考情報」のセクションを置くこと。
- 出典は「- [タイトル](URL)」形式の箇条書きで 1 件以上列挙すること。
- 一次情報(公式リリース、公式ドキュメント、CVE/NVD、GitHub release 等)の出典には末尾に「（一次情報）」を付けること(1 件以上必須)。それ以外の出典にはラベルを付けない(「（二次情報）」とは書かない)。
- 公式リリース、公式ドキュメント、CVE/NVD、GitHub release などの一次情報を優先すること。
- セクション内に「情報確認日: %s」という行を 1 行置くこと。
- 出典に基づく主要な事実・数値・発表内容を本文で述べる箇所では、その語句に「[語句](URL)」形式のインラインリンクを付けること。読者が本文中でどの記述がどの出典に基づくかを追えるようにする。
- 本文でインラインリンクとして引用した URL は、末尾の参考情報一覧にも必ず載せること。
- [リサーチ結果] が提示されている場合、そこに含まれる出典 URL を参考情報セクションに反映すること。
