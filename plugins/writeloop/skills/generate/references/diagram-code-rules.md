## common
- すべてのコードブロックの開始フェンスに言語タグを付ける（例: ```go、```mermaid。プレーンテキストは ```text）。
- コードブロックの直前に「（コード種別: 実行可能）」「（コード種別: 疑似コード）」「（コード種別: 概念例）」のいずれかのラベル行を置くことを推奨する。
