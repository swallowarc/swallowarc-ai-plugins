# 一枚絵インフォグラフィック生成プロンプト（{{主題}}）

`{{元になった素材}}` の内容を、手描き風ホワイトボード解説イラスト1枚にまとめるための画像生成プロンプト。

## 使い方のコツ（重要）

- 下のプロンプトの **「描く文字」は一字一句そのまま** 渡す。抽象指示だと英語混入・文字化けの原因。
- 末尾の **「指定した文字以外は描かない」** 指示を消さない。
- 「日本語で」などの余計な語は足さない。
- 文字が潰れる場合は、ゾーンを減らすか「コンパクト版」を使う。

> 注：図中の文言はすべて素材の内容に基づく。創作の主張・比喩は入れていない。

```text
手描きのマーカースケッチ風インフォグラフィック。クリーム色の紙の背景、ラフな角丸の枠線、
蛍光ペンの下線、やわらかいパステル配色、手書き風の日本語フォント、かわいい丸いロボットの
マスコット。横長16:9、情報は左から右へ流れる構成、余白をとって読みやすく明るいトーン。

【上部：タイトル帯】
大きな手書き見出し「{{タイトル}}」、その下に小さく「{{サブタイトル}}」。
（必要なら）付箋・クリップメモに「{{出典・補足}}」。

【中央：主役】
見出し「{{核メッセージの見出し}}」。{{中央に描く関係・キャラ・矢印の説明}}。
表示文字：{{中央ゾーンのラベルを全列挙}}

【{{周辺ゾーン1}}】{{説明}}。表示文字：{{ラベル列挙}}
【{{周辺ゾーン2}}】{{説明}}。表示文字：{{ラベル列挙}}
（必要なゾーンだけ。4〜6個まで）

画像に描く文字は上記で挙げたものだけ。指定した文字以外の文字・ラベル・英単語は描かない。
漢字・ひらがな・カタカナを正確に、文字化けや誤字なく描く。
```

## GPT Image 2 版（制作ブリーフ形式）

```text
{{主題}}の要点をまとめた、手描きホワイトボード解説風のインフォグラフィックを横長1枚で作成してください。
全体は手描きのマーカースケッチ調で、クリーム色の紙背景、ラフな角丸枠、蛍光ペンの下線、
パステル配色、手書き風の日本語フォント、かわいい丸ロボットのマスコット、明るく親しみやすいトーン。
情報は左から右へ流れる構成で、余白をとって読みやすく。

上部に手書き見出し「{{タイトル}}」、その下に「{{サブタイトル}}」。（必要なら付箋に「{{出典}}」）。
中央を主役にして「{{核メッセージの見出し}}」を置き、{{中央の説明と表示文字を散文で}}。
{{周辺ゾーンを散文で。各ゾーンの表示文字を明記}}。

画像に描く文字は、ここで挙げた日本語の文言だけにしてください。指定していない文字・英単語は描かないでください。
漢字・ひらがな・カタカナは正確に、文字化けや誤字なく描いてください。
```

API で比率を指定する例：`size: "1536x1024"`（横長）, `quality: "high"`。

## コンパクト版（文字を減らして確実に読ませたいとき）

```text
{{主題}}を、手描きマーカースケッチ調のインフォグラフィックで横長1枚にまとめてください。
クリーム色の紙背景、ラフな角丸枠、蛍光ペンの下線、パステル配色、手書き風日本語フォント、
かわいい丸ロボット。中央に大見出し「{{核メッセージ}}」。{{最小限の周辺要素}}。
描く文字は上記の日本語だけ。指定外の文字は描かない。文字化けなく正確に描いてください。
```

## 配色メモ（任意・素材と合わせる場合）

- {{意味づけと16進カラー。例：流れ＝青 #3a5a8c／対比＝赤紫／成果＝緑}}

## 仕上げの調整

- 文字が崩れた箇所は、その部分だけ**部分再生成（インペイント）**で直す。GPT は会話で修正。
- 情報過多で潰れる場合はゾーンを削る／コンパクト版を使う。
- シリーズ化（章扉など）はスタイルを固定し、中央ゾーンのテーマだけ差し替える。
