# スタイルガイド（手描き説明画像）

このスキルが目指すのは、YouTube 等でよく見る**ゆるかわ手描きホワイトボード解説風**の1枚絵インフォグラフィック。

## ビジュアルの骨子

プロンプトに必ず織り込む要素：

- **画風**：手描きのマーカースケッチ調（doodle / whiteboard explainer）
- **背景**：クリーム色の紙、軽い紙テクスチャ
- **枠・線**：ラフな角丸の枠線、少し不揃いの手描き線
- **強調**：キーワードに蛍光ペン（マーカー）の下線
- **配色**：やわらかいパステル。色は意味づけに使う（後述）
- **文字**：手書き風の日本語フォント
- **キャラ**：かわいい丸いロボット等のマスコット（色違いで複数体も可）
- **装飾**：吹き出し、付箋、クリップ留めメモ、軽いキラキラ、矢印
- **構図**：横長（16:9 目安）、情報は**左から右へ流れる**、余白をとって読みやすく、明るく親しみやすいトーン

## ゾーン例（内容に合うものを選ぶ／全部は使わない）

| ゾーン | 用途 | 表現 |
|---|---|---|
| タイトル帯（上部） | 主題・出典 | 大きな手書き見出し＋付箋やクリップメモで補足（出典・サブタイトル） |
| 中央の主役 | 核メッセージ1つ | 最も大きく。キャラ＋矢印で関係を見せる |
| 左→中→右のフロー | プロセス・流れ | ノード＋矢印で左から右へ |
| 対比（before/after, A/B） | 問題と解決、二項対立 | 2枚のカードを並べ、間に矢印や VS |
| 指標カード | 増減・数値 | 小カードに数値＋↑↓ |
| 手順ステップ | 段階・順序 | 番号バッジ付きの縦/横ステップ |
| ロードマップ | 章・パートの並び | 道や立て札に番号付きラベル |
| マスコット注釈 | 立場・役割・読者 | 小ロボット＋吹き出しラベル |
| 下部キャプション | 一言まとめ | 細い手書き文で全体を要約 |

## 配色の意味づけ（任意）

- 流れ・前進＝青系、警告・対立＝赤系、肯定・成果＝緑系、補足＝灰/ベージュ。
- 元になった資料（HTML 等）に配色があるなら、その**アクセント色を引き継ぐ**と一貫する。
- 色コードを指定したいときはプロンプトに16進で添える（例：青 `#3a5a8c`）。モデルは厳密一致しないが方向づけにはなる。

## レイアウト設計のコツ

- **主役は1つ**。中央に最も大きく置く。周辺は主役を支える情報に絞る。
- ゾーンは **4〜6個まで**を目安に。多いと文字が潰れる。
- 関係性（流れ・対比・順序）は**矢印・並び・番号**で視覚化する。
- 文字量が多いと判断したら、ゾーンを削るか「コンパクト版」を別に用意する。
